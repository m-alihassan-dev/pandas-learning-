# 📊 Pandas Learning

A collection of Jupyter notebooks for learning and practicing `pandas` — the Python library for data analysis and manipulation.

## 📂 Notebooks

Notebook | Description
--- | ---
`pandas_series.ipynb` | Introduction to pandas `Series` and basic operations
`pandas_dataframes.ipynb` | Working with pandas `DataFrame` structure and creation
`pandas_operations.ipynb` | Core dataframe operations, arithmetic, and transformations
`conditional_selections.ipynb` | Boolean indexing and conditional filtering
`concatenation.ipynb` | Concatenating and combining pandas objects
`joining.ipynb` | Joining dataframes with inner, outer, left, and right joins
`merging.ipynb` | Advanced dataframe merging using keys and indexes
`groupby_aggregation.ipynb` | Grouping data and applying aggregation functions
`pivot_tables.ipynb` | Pivot tables, reshaping data, and cross-tabulation
`missing_data.ipynb` | Detecting, cleaning, and imputing missing values
`feature_extraction_project.ipynb` | Example feature extraction workflow with sample data
`countries.ipynb` | Country dataset exploration and practical examples

## 🚀 Getting Started

### Prerequisites
- Python 3.8+ installed
- Jupyter Notebook or JupyterLab

### Installation

```bash
git clone https://github.com/your-username/pandas-learning.git
cd pandas-learning
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Launch Jupyter

```bash
jupyter lab
```

or

```bash
jupyter notebook
```

Open the notebook you want to explore.
